# 📊 Retail Sales Forecasting & Demand Analysis

## Overview

This project analyzes historical retail sales data to forecast future sales, detect anomalies, segment product demand, and provide business insights using Machine Learning and Time Series Forecasting.

The project was developed as part of my internship and includes an interactive Streamlit dashboard for visualization and forecasting.

---

## Features

- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Time Series Analysis
- Sales Forecasting using:
  - SARIMA
  - Prophet
  - XGBoost (Best Performing Model)
- Product Category & Region-wise Forecasting
- Anomaly Detection using:
  - Isolation Forest
  - Rolling Z-Score
- Product Demand Segmentation using K-Means Clustering
- Interactive Streamlit Dashboard
